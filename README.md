### O que eu aprendi

Neste projeto foi melhorado as habilidades de css, tendo ênfase na propriedade flex.


``` css
body{
    min-height: 100vh;
    min-width: 100vw;
    display: flex;
    justify-content: center;
    align-items: center;
    background-color:hsl(217, 54%, 11%);
    font-family: 'Outfit', sans-serif;
}
```
